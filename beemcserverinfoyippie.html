Hi! My minecraft server has a few mods. You can download a zip file with all the mods bundled here:

Download mod pack

The mods included are Ribbits (fucking adorable frog villagers), the glunch (slightly altered to have much less destructive potential), a minimap mod, and customizable player models which you can use to change how your little minecraft guy looks. It's totally optional to customize your model so you don't have to if you don't want to, but if you do want to there's a bunch of tutorials on youtube on how to use it.

You'll need Minecraft version 1.20.1 with Forge installed. If you don't have Forge installed you can find the 1.20.1 version here.
